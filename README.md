# TTS Portal

A local text-to-speech web app with voice design and voice cloning.

## Setup

```bash
uv sync
uv run app.py
```

Open http://localhost:5001 in your browser.

## Features

- **Voice Design** — describe the voice you want (age, accent, tone)
- **Voice Cloning** — upload a reference audio clip to clone a speaker
- Multi-language support (English, Chinese, Japanese, Korean, and more)
- Generation history with playback and download
